# PortIO / 联合映射

## 快速启动

1. 本地安装 [Node.js](https://nodejs.org), [yarn](https://yarnpkg.com/) 及 [Git](https://git-scm.com/)
2. 克隆本仓库 `git clone https://github.com/kingc2022/PortIO.git`
3. 使用 `yarn install` 安装依赖
4. 使用 `yarn dev` 启动开发服务器
